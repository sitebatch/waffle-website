# Supported Libraries

Waffle supports the following libraries. Depending on the library you are using, use the appropriate wrapper function.

| Library | Prevention | Example and Documents |
|:--- |:--- |:--- |
| net/http | Common Web Attacks<br />SSRF | [contrib/net/http](https://github.com/sitebatch/waffle-go/tree/main/contrib/net/http) |
| gin-gonic/gin | Common Web Attacks  | [contrib/gin-gonic/gin](https://github.com/sitebatch/waffle-go/tree/main/contrib/gin-gonic/gin) |
| labstack/echo | Common Web Attacks | [contrib/labstack/echo](https://github.com/sitebatch/waffle-go/tree/main/contrib/labstack/echo) |
| 99designs/gqlgen | Common Web Attacks | [contrib/99designs/gqlgen](https://github.com/sitebatch/waffle-go/tree/main/contrib/99designs/gqlgen) |
| os | Sensitive file open | [contrib/os](https://github.com/sitebatch/waffle-go/tree/main/contrib/os) |
| database/sql | SQL Injection | [contrib/database/sql](https://github.com/sitebatch/waffle-go/tree/main/contrib/database/sql) |
| gorm.io/gorm (v2) | SQL Injection | [contrib/gorm.io/gorm](https://github.com/sitebatch/waffle-go/tree/main/contrib/gorm.io/gorm) |
